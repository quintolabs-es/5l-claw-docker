# Project Skills

Add repo-specific skills here.

- Each skill is a folder, not a single markdown file.
- Each skill folder must contain `SKILL.md`.
- Skills must follow the Agent Skills packaging specification:
  https://agentskills.io/specification
- Skills are copied to the project `.aie-os` folder and referenced in the aggregated context file.
- `README.md` is ignored by build.

Example:

```text
project-skills/
  add-endpoint/
    SKILL.md
```
